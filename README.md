## セットアップ

サーバー・ビルド不要。

1. リポジトリを取得（またはZIP展開）
2. `index.html` をブラウザで開く（PC: Chrome/Edge、iPad: Safari）

配布はHTMLファイルの直接共有（Google Drive等）で行う。GitHub Pages公開は未確定（要件定義書§9 No.6）。
